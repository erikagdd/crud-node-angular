1. Install:
-----------------------------------------------------------------------------------------
    $ git clone https://github.com/callicoder/node-easy-notes-app.git CRUD_Express_Mongo_Postman
    $ cd CRUD_Express_Mongo_Postman/
    $ npm install
    $ sudo service mongod start
    $ node server.js
    - http://localhost:3000

2. MongoDB:
-----------------------------------------------------------------------------------------
    - Obrim una altra terminal en Visual Studio per crear la db, la collection i crear una note
    $ mongosh
        > show dbs
        > use easy-notes
        > db.notes.insert( { "title": "title1", "content": "content1"})
        > show collections
        > db.notes.find().pretty()
        > db.dropDatabase();

3. Postman:
-----------------------------------------------------------------------------------------  
    - POST http://0.0.0.0:3000/notes
        {
            "title": "title_1",
            "content": "content_1"
        }
    - GET http://0.0.0.0:3000/notes
    - GET http://0.0.0.0:3000/notes/62861f8e919cef6082cecd29
    - PUT http://0.0.0.0:3000/notes/62861f8e919cef6082cecd29
        {
            "title": "title_2",
            "content": "content_2"
        }
    - DELETE http://0.0.0.0:3000/notes/62861f8e919cef6082cecd29

4. Crear una col.lecció i la documentació de l'anterior API creada a l'exercici de CRUD, en Postman:
